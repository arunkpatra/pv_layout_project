## What was built

### 1. Semantic token system — both themes, one file
- [`docs/design/tokens/tokens.css`](../design/tokens/tokens.css) — canonical source. Light values on `:root`, dark values on `[data-theme="dark"]`. Every component in the mocks consumes variables from this file; no color literals, no pixel literals that could have been tokens.
- Covered: surfaces, text, borders, accent, signal, data-vis canvas palette, typography (family/scale/weight/line-height/tracking), spacing (4px grid), sizing (control/rail/bar/inspector), radius, elevation, motion.

### 2. Shared base component CSS

## Known limitations / decisions to flag

1. **Dark theme is draft-only.** `pv-dark.json` MapLibre style not authored yet (S8). Two flagged candidates for S13.5 polish: ICR footprint contrast vs. surrounding tables in populated canvas; secondary button lift in startup dialog. Expected; not blockers.
2. **Fonts are fallback-only** in the mocks. System-Inter on macOS renders close to bundled Inter; Geist Mono fallback stack. Real woff2 bundling lands in S6 at `packages/ui/assets/fonts/`. Mocks render acceptably either way.
3. **MapLibre vector styles not authored.** The canvas features in the populated mock are approximations of what `pv-light.json` will render in S8. The canvas visual language spec in DESIGN_FOUNDATIONS §8 is the contract S8 implements against.
4. **Brand identity is placeholder.** Wordmark glyph is the Lucide `sun` icon — same glyph the S5 Tauri build ships, so mocks and running app agree today. Real brand (logo, icon, accent, wordmark) lands in S13.6.
5. **Solar-specific custom icons are inventory only.** Named and scoped in DESIGN_FOUNDATIONS §9, but not drawn. They land in S6 when the design system is implemented in code.

---

## Decisions made during this spike

- **Render method:** static HTML → Playwright-rendered PNG (over SVG or Figma). Rationale: single source of truth, reproducible, tokens share the same stylesheet the real app will use.
- **Five mocks, not four.** Added Splash as the fifth so the cold-start experience is designed, not improvised.
- **Dark produced via `data-theme` flip** on the same HTML, not separate dark HTML. Keeps the token system honest — if a component doesn't flip cleanly, that's immediately visible as a bug, not a new authoring task.
- **Render DPR is 1× (1440×900).** Previously 2× (2880×1800), which exceeded Anthropic's many-image dimension limit and crashed the prior session. DPR ≤ 1.38 stays safe on this viewport. Patched in `render-mocks.mjs`.
- **Accent is `#d36e31`** placeholder (warm amber, Claude-Desktop neighborhood). Real accent arrives in S13.6.
- **Wordmark is a Lucide sun**, not an invented logo. Matches the S5 Tauri shell build. Real logo arrives in S13.6.
